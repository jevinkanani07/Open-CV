# Video Capture using OpenCV

This folder contains examples of capturing and displaying video
from a webcam or video file using OpenCV.

## What you will learn
- Concept of video capture
- How to use cv2.VideoCapture()
- Reading video from webcam or file
- Displaying video frame by frame
- Properly closing video windows

## Files
- st_video.py : Basic example of starting video from webcam or file

## Use case
Video capture is widely used in real-time face detection,
object tracking, and computer vision applications.
