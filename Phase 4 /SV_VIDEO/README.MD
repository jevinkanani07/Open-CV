# Saving Video using OpenCV

This folder contains examples of recording video
from a webcam or frames and saving it into a video file using OpenCV.

## What you will learn
- Concept of video recording
- How to use cv2.VideoWriter()
- Setting frame size and FPS
- Understanding video formats (.avi, .mp4)
- Properly closing the recording

## Files
- Save_Video.py : Basic example of recording and saving video

## Use case
Video saving is used in surveillance systems,
data collection, computer vision projects,
and real-time applications.
