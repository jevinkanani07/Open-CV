# Median Blur using OpenCV

This folder contains examples of applying median blur on images using OpenCV.

## What you will learn
- Concept of median blur
- How to use cv2.medianBlur()
- Effect of kernel size
- Basic idea of noise reduction

## Files
- median_blur.py : Basic example of applying median blur on an image

## Use case
Median blur is commonly used for removing salt-and-pepper noise,
image smoothing, and preprocessing in computer vision tasks.
