# Image Blurring using OpenCV

This folder contains examples of blurring images using OpenCV.

## What you will learn
- Concept of image blurring
- How to use cv2.blur()
- How to use cv2.GaussianBlur()
- Effect of kernel size

## Files
- gaussian.py : Basic image blurring example

## Use case
Image blurring is used for noise reduction,
privacy protection, and
preprocessing in computer vision tasks.
