# Interactive Drawing on Image using OpenCV

This folder contains an interactive OpenCV program
that allows the user to add lines, rectangles, circles,
and text to an image.

## What you will learn
- Image path validation using cv2.imread()
- Drawing selection through user input
- Drawing line, rectangle, and circle
- Adding custom text on images
- Font selection and BGR color handling
- Saving the final edited image

## Features
- Supports Line, Rectangle, Circle, and Text
- Custom color input (B G R)
- Multiple font options for text
- Automatically saves the edited image

## Files
- All in one.py Interactive image editing script

## Use case
This program is useful for learning OpenCV drawing functions,
basic image annotation,
and understanding computer vision fundamentals.
