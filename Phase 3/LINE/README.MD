# Drawing Line using OpenCV

This folder contains examples of drawing lines on images using OpenCV.

## What you will learn
- Concept of drawing lines
- How to use cv2.line()
- Understanding start and end points
- Effect of thickness and color

## Files
- draw_line.py : Basic example of drawing a line on an image

## Use case
Drawing lines is used for boundary marking,
object detection visualization, and image annotation.
