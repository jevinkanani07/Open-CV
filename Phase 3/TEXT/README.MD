# Putting Text on Image using OpenCV

This folder contains examples of adding text to images using OpenCV.

## What you will learn
- Concept of putting text on images
- How to use cv2.putText()
- Understanding font type, size, and color
- Effect of thickness and position

## Files
- draw_text.py : Basic example of putting text on an image

## Use case
Adding text to images is used for labeling,
object identification, and computer vision visualization.
