# Drawing Rectangle using OpenCV

This folder contains examples of drawing rectangles on images using OpenCV.

## What you will learn
- Concept of drawing rectangles
- How to use cv2.rectangle()
- Understanding top-left and bottom-right points
- Effect of thickness and color

## Files
- draw_rectangle.py : Basic example of drawing a rectangle on an image

## Use case
Drawing rectangles is commonly used for object detection,
face detection bounding boxes, and image annotation.
