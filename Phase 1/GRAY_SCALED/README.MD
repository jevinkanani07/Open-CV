# Grayscale Image using OpenCV

This folder contains examples of converting a color image to grayscale using OpenCV.

## What you will learn
- What is a grayscale image
- How to use cv2.cvtColor()
- Conversion from BGR to GRAY color space

## Files
- Gray_Scaled.py : Example of converting a color image to grayscale

## Use case
Grayscale images are widely used in face detection, edge detection,
and to simplify and speed up image processing tasks.
