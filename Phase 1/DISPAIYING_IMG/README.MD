# Displaying Image using OpenCV

This folder contains examples of displaying images using OpenCV.

## What you will learn
- How to use cv2.imshow()
- Opening an image window
- Purpose of cv2.waitKey() and cv2.destroyAllWindows()

## Files
- display_image.py : Basic image display example

## Use case
To visualize the result of image processing or face detection,
displaying the image on screen is required.
