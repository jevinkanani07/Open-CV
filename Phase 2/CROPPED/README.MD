# Image Cropping using OpenCV

This folder contains examples of cropping a specific region of an image using OpenCV.

## What you will learn
- Concept of image cropping
- Using NumPy slicing
- What is ROI (Region of Interest)

## Files
- crop_image.py : Basic image cropping example

## Use case
Image cropping is used in face detection,
object detection, and removing unwanted areas from images.
