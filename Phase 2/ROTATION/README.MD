# Image Rotation using OpenCV

This folder contains examples of rotating images using OpenCV.

## What you will learn
- Concept of image rotation
- How to use cv2.getRotationMatrix2D()
- How to use cv2.warpAffine()
- Effect of angle and scale

## Files
- rotation.py : Basic image rotation example

## Use case
Image rotation is commonly used in data augmentation,
image alignment, and computer vision tasks.
