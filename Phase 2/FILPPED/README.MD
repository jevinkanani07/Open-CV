# Image Flipping using OpenCV

This folder contains examples of flipping images using OpenCV.

## What you will learn
- Concept of image flipping
- How to use cv2.flip()
- Horizontal, vertical, and both flip types

## Files
- Flipping.py : Basic image flipping example

## Use case
Image flipping is widely used in data augmentation,
mirror effects, and improving computer vision models.
