# Image Resizing using OpenCV

This folder contains examples of resizing images using OpenCV.

## What you will learn
- Concept of image resizing
- How to use cv2.resize()
- Using width, height, and scale factor
- Basic idea of interpolation methods

## Files
- resize.py : Basic image resizing example

## Use case
Image resizing is used for model input preparation,
display purposes, and storage optimization.
